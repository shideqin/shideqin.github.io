安装mysql5.7社区版二进制软件包出现 error while loading shared libraries: libaio.so.1:

发表于 2017-03-11 | 分类于 Mysql

/usr/local/mysql/bin/mysql_install_db –basedir=/usr/local/mysql –datadir=/usr/local/mysql/data –user=mysql 在安装二进制mysql的软件包时（版本mysql5.7社区版），初始化中出现 error while loading shared libraries: libaio.so.1: cannot open shared object file: No such file or directory的错误： 解决方法： [shell] $ yum install -y libaio //安装后在初始化就OK了 [/shell]
